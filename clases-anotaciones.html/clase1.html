Clase 1 - Introducción a REACT

React, es una librería de Javascript

En React ya no dividimos nuestras carpetas por tecnologías, sino mas bien, por funcionalidad.

SPA : Single Page Aplicattion

En un proyecto de React, voy a tener en el mismo lugar, la parte logica y la parte visual. Esto es gracias a JSX (notacion de JS que permite integrar html y JS)

React, reacciona... y se actualiza, totalmente dinamico.

React crea 2 DOM, esta el DOM y el VIRTUAL DOM

Cada vez que hacemos cualquier actualizacion se vuelve a cargar todo de vuelta en el VIRTUAL DOM (interno, no se ve) y lo que hace REACT es comparar (Reconciliacion), para ver que cambio (numero, letra, lo que sea) internamente renderiza todo, no lo hace visual, no consume tantos recursos del equipo, te carga todo con el cambio que haya ocurrido unicamente.

JSX : "Sintaxis mezclada entre HTML y JS.

Cuando trabajamos con JSX, hay un proceso que se llama transpilación, que es como la compilacion que pasa de un lenguajue a otro. (Esto se hace automaticamente con React APP o con Vite, pasa de JSX a Html y JS)

Render : Renderizo, "cuando algo lo mostramos en el DOM"
